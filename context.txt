## Overview
PayFlow is a B2B payroll SaaS for African businesses (Nigeria, Ghana).
Handles employee management, payroll calculation, tax compliance, wallet/payments, and HR functions.

## Tech Stack
- Backend: Go 1.26, Chi v5 router, GORM ORM, PostgreSQL, Redis, Asynq
- Frontend: React 18, TypeScript, Vite, Tailwind CSS, TanStack Query, Zustand
- Email: Brevo SMTP via Asynq async queue
- Payments: Paystack, KoraPay, VFD
- Deployment: Railway (backend), Vercel (frontend)

## Current Migration Version: 000027

## Features
- Multi-country payroll (Nigeria PAYE/Pension/NHF/NSITF, Ghana SSNIT)
- Payroll lifecycle: create > submit > approve > process > complete
- Multi-cadre salary structures with custom deduction rules
- Employee management with CSV import and BVN verification
- Business wallet with virtual accounts (KoraPay)
- Deposits via Paystack Checkout (card, bank, USSD)
- Double-entry accounting ledger with daily + weekly reconciliation
- Transfer system with multi-provider fallback and circuit breaker
- Leave management (types, requests, approval, balances)
- Loan management with automatic payroll deductions
- Employee self-service portal (separate auth + layout)
- Org-specific provider key overrides (AES-256 encrypted)
- Billing plans (Free/Growth/Enterprise via Paystack)
- Platform admin (super admin dashboard, org management)
- Redis cache for cadres, deductions, wallet balance
- Read replica support for report queries
- Async payroll processing with status polling
- Velocity controls and anomaly detection

## API: 90+ endpoints across Auth, Employees, Cadres, Payroll, Wallet, Leave, Loans, Transfers, Billing, Platform

## Frontend: 50+ pages including admin dashboard, employee self-service portal, and public landing page

## Tests: Integration (testcontainers), Security, E2E, Load (k6), Frontend (Vitest)

## Background Jobs: Email delivery, Payroll processing, Daily reconciliation, Weekly provider reconciliation
